This repository contains a Food Tech mobile application prototype designed in Figma. The app focuses on providing a smooth and user-friendly experience for browsing restaurants, ordering food, and tracking deliveries.
The prototype showcases modern UI design, intuitive navigation, and seamless user flows for an enhanced food ordering experience.

🎯 Key Features
🔍 Browse restaurants and food categories
🍕 View detailed food items with pricing
🛒 Add to cart and manage orders
💳 Checkout and payment flow
📍 Order tracking interface
👤 User profile section
🎨 Design Highlights
Clean and modern UI
Mobile-first responsive layout
Consistent color palette and typography
Easy-to-use navigation flow
